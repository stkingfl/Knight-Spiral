Rules

Each player scans the square spiral from their last position and places the next piece on the first empty square not attacked by an opponent.

Things To Notice

  • The default two-knight run recreates the recognizable OEIS A392177 territory split.
  • Changing queue order changes who claims boundary cells first, even with the same pieces.
  • The public palette currently focuses on leapers so long runs stay responsive.

Things To Try

  • Swap the second Knight for a Camel and watch the fronts shear apart.
  • Use three or more leapers to turn the two-color bands into competing strata.
  • Turn on Indexes at high zoom to inspect the actual spiral order.
